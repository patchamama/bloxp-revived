Advanced configuration

First of all, be aware that in order to use the Advanced configuration it may be necessary to have some basic knowledge of HTML and web design. I would only recommend to use below form in cases where the basic one at the home page is not working or for taking advantage of more flexibility to select which posts to export.

Please, before using the Advanced configuration read carefully all help notes below the form.

Having said this, with Advanced configuration you will be able to:

  • Export all posts from a blog, even if there are more than 250, in separated volumes (e.g. export 750 posts out of a blog in three volumes of 250 posts each)
  • Help Bloxp at posts' link detection if it's unable to do it automatically
  • Start exporting from a different post than the newest one

Configuration help

Please, make sure you understand the following notes in order to properly set the advanced configuration.

SET FIRST POST

First post URL:
This is the URL link to the first post you want to export. Normally Bloxp will start from this post and will continue in reversed chronological order (can be changed with manual EXPORT SETTINGS).

E.g. If there are 50 posts in total and you start at post #10, you will export from #10 to #1 and you won't export the 40 oldest posts.

First post title:
This is the title of the first post. To get the best results you should visit the URL written in the previous step and copy the content of the title bar shown by your browser (removing the main site title from it if necessary).

E.g. The title for this URL (www.bloxp.com/advancedConf.php) is "Bloxp, advanced configuration". Then the post title should be "advanced configuration" (note I have removed "Bloxp, " and left "advanced configuration" without changes)

SITE INFORMATION

Fields under this section should contain:

  • Main site URL, e.g. "http://www.bloxp.com"
  • Main site name, e.g. "Bloxp"
  • Short site description, e.g. "Use Bloxp to export any blog and turn it into a downloadable ebook! Save your favorite blog as a unique, readable file."

EBOOK OPTIONS

These options can be enabled or disabled depending on the desired result:

  • Transform links to footnotes, if checked, links will be shown as footnotes for each post
  • Add Table of Contents, if checked, a TOC will be added to the eBook

EXPORT SETTINGS

Manual export settings should be used only if completely necessary. I recommend to try first with default export settings and if Bloxp is not working then proceed to change them manually.

Bloxp tries to export all the blog posts by jumping from one post to the previous one until no more posts are found. This is done by searching for the link to its previous post. If the link doesn't exist, Bloxp won't be working at all. However, sometimes the link exists but Bloxp is unable to find it automatically. In these cases you can help the system by applying manual export settings.

  1. Open in one tab the URL for the first post you have set at the beginning
  2. By pressing Ctrl+U (or Cmd+Opt+U) you will open the Source Code for that page
  3. By pressing Ctrl+F (or Cmd+F) you are able to Search within the page
  4. Insert in the search bar the URL of the previous post (e.g. for post #9 insert URL to post #8)
  5. If you cannot find it then unfortunately you won't be able to use Bloxp. Otherwise, continue with next step!
  6. Fill the export settings based on the link code.
  7. Done!

To help you understand the export settings, below are some samples of how to fill them (always without quotes!). Remember that in order for the settings to work properly, the link found should have the same schema in all posts. This means that links under "Recommended posts" or "Most read" widgets shouldn't be used.

Sample 1:
<link rel="prev" href="http://www.bloxp.com/previous/post" />

  • Tag name, "link"
  • Attribute name, "rel"
  • Attribute value, "prev"
  • Prepend string, ""
  • It's parent tag, unchecked

Sample 2:
<a class="blog-pager-older-link" href="http://www.bloxp.com/previous/post" />

  • Tag name, "a"
  • Attribute name, "class"
  • Attribute value, "blog-pager-older-link"
  • Prepend string, ""
  • It's parent tag, unchecked

Sample 3:
<a class="blog-pager-older-link" href="/previous/post" >

  • Tag name, "a"
  • Attribute name, "class"
  • Attribute value, "blog-pager-older-link"
  • Prepend string, "http://www.bloxp.com"
  • It's parent tag, unchecked

Sample 4:
<span id="previous_post"><a href="http://www.bloxp.com/previous/post" >Previous post</a></span>

  • Tag name, "span"
  • Attribute name, "id"
  • Attribute value, "previous_post"
  • Prepend string, ""
  • It's parent tag, checked

Answers


Why some blogs are not working with Bloxp?

Bloxp always tries to find the links that connect one post with its previous so it can somehow go back in the blog history and create an index. If for any reason this link cannot be found Bloxp will be unable to crawl the entire site.

Go up

I'm writing a correct URL but it's still not working. What can I do?

First of all, check if the given link is not redirecting to a second URL. In that case, please try writing the original link. Also, be sure that the website has a feed/rss.

Go up

Why Bloxp has a limit of 250 posts?

The main reason is because the server where Bloxp is running is not as powerful as I would like. Also, the generated ebook filesize could be too big depending on the number of images and their resolution.

If you need to export a blog with more than 250 posts, please contact me.

Go up

What can I do if Bloxp is not working with my site?

You should modify (or ask to) your HTML code adding in your posts' heading (between <head> and </head>) the following tag:

<link rel="prev" href="Previous post URL" />

Go up

How can I improve the way Bloxp processes my content?

In this link you will find useful tips for increasing your articles' readability. Consider also that without enough text in an article (e.g. a post with only pictures) Bloxp won't be able to find the relevant content.

I don't want Bloxp being used with my site. Can I block it?

If you prefer not to allow people to export your contents as an ebook you just have to send me an email to contact@bloxp.com including the URL you want to block. I'll try to attend your petition as fast as possible (usually within 24-48h).

What does "Session has expired" mean?

Every time you start a new ebook a session is created and all data collected is kept with a reference to it. If you spend more than 15 min without interacting with Bloxp this session will expire and all the information will be lost. If this happens you will have to start again from the beginning.

Go up

I've forgotten to check the "Add Table of Contents" option, can I add it without starting from the Homepage?

I'm sorry but no at this moment. I hope this will be added/fixed soon! The same applies to the "Transform links to footnotes" option.

Go up

Is there any option to save an ebook as a PDF?

At this moment Bloxp cannot create PDFs. However, you can easily save an ebook in this format with just two clicks:

(1) Once the blog you want to export has been indexed, click on the "Preview" link (eye-shaped icon next to the Exporting options).

(2) Use your Web Browser's options to print the webpage as a PDF!

Go up

What email should I use to receive an ebook in my Kindle?

Using an email such as [name]@kindle.com your new ebook will be sent over 3G. However, this option can incur charges on your Kindle account. If prefered, use [name]@free.kindle.com to receive your ebook via Wi-Fi when connected. This option is free in all cases.

For more information please refer to Amazon.

My new ebook has been sent but I cannot find it in my Inbox! Where is it?

First of all, please check that the mail is not in the Spam folder. If still you cannot find it, the problem could be that the attached file is too big for your Inbox. In this case, make sure you have enough free space or try downloading the ebook directly from Bloxp.
